代码分享工具

发表于 2019-06-21 | 分类于 工具 | 阅读次数 ℃
本文字数： 475 | 阅读时长 ≈ 1 分钟

身为IT人士，代码分享在日常工作中实在是再寻常不过了。比起发送文件的方式，一个代码分享链接工具是再好不过的了！简便又快捷。尤其在寻求帮助的时候，大段大段代码发在群里实在让人难以接受，而且还有被聊天工具转义的风险。

首推工具

  • https://pastebin.com/

pastebin 工具非常强大，支持多种语言、过期时间、是否加密、代码下载、在线打印、新旧文件参照对比等等，部分功能需要登录使用。另外，谷歌插件版用起来更方便哦！

小巧实用

  • https://paste.ubuntu.com/
    ubuntu 社区推出的分享工具，支持语言选择、过期时间、文件下载。界面简洁，无需注册。

大材小用

  • https://gist.github.com/
    Github 旗下工具，功能也很强大，有点像单独分支，只是单纯用来分享代码觉得有点大材小用了。

极致精简

  • https://hastebin.com
    非常精简的代码分享工具

中规中矩

  • http://pasted.co/
    代码分享倒不是很友好，文字分享倒觉得还可以吧

PS：工具不在多，选择适合自己的就好！
